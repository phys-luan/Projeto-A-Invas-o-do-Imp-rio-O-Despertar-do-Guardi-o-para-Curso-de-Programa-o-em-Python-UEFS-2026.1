# 🎮 O Despertar do Guardião

Jogo desenvolvido em **Python** utilizando a biblioteca **Pygame**.

## 📖 Sobre o Projeto

**O Despertar do Guardião** é um jogo de ação 2D inspirado no universo Star Wars. O jogador controla uma nave espacial e deve sobreviver a ondas de inimigos enquanto acumula pontos. Durante a partida é possível coletar bactas para recuperar vida e enfrentar o chefe final, **Darth Vader**.

Projeto desenvolvido por **Luan Almeida Meira**.

---

## ✨ Funcionalidades

* 🚀 Controle de nave espacial.
* 🔫 Sistema de tiros.
* 👾 Inimigos Stormtroopers.
* ❤️ Coleta de bactas para recuperar vida.
* 🏆 Sistema de pontuação.
* ⚙️ Três níveis de dificuldade:

  * Fácil
  * Médio
  * Difícil
* ⏸️ Menu de pausa.
* 🔊 Efeitos sonoros e músicas.
* 👑 Batalha final contra Darth Vader.
* 🎉 Tela de vitória e Game Over.

## ▶️ Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/o-despertar-do-guardiao.git
```

### 2. Entre na pasta do projeto

```bash
cd o-despertar-do-guardiao
```

### 3. Instale o Pygame

```bash
pip install pygame
```

### 4. Execute o jogo

```bash
python main.py
```

## 🎮 Controles

| Tecla  | Função              |
| ------ | ------------------- |
| W / ↑  | Mover para cima     |
| S / ↓  | Mover para baixo    |
| A / ←  | Mover para esquerda |
| D / →  | Mover para direita  |
| Espaço | Atirar              |
| ESC    | Pausar jogo         |

---

## 🏆 Objetivo

* Eliminar Stormtroopers para ganhar pontos.
* Coletar bactas para recuperar vida.
* Sobreviver o máximo possível.
* Alcançar 200 pontos para enfrentar Darth Vader.
* Derrotar Darth Vader para vencer o jogo.
